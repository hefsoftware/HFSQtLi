Single row query execution

The following function allows easy operation on any query that returns one and only one row. They perform in one operation binding of parameters, a step on first row, fetching the data of the row and a second step to check the query effetively returned only one row. For example:

int x;
qry.executeSingle<1>(4, x);

Is more or less equivalent to:

qry.bind(4) && qry.step() && qry.columnAll(x) && "qry.step() returns SQLITE3_DONE"

Note All the function in this group will return an error if the query returns no row or more than one row

Commands query execution

The following function allows easy operation on any query that returns no row (e.g. CREATE TABLE or DELETE) They perform in one operation binding of parameters, and a step to run a query, checking that effectively the query returns no rows For example:

qry.execute(4);

Is more or less equivalent to:

qry.bind(4) && "qry.step() returns SQLITE3_DONE"

Note All the function in this group will return an error if the query returns any row.
